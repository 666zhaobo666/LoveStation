# ❤️ LoveStation 

一个专为情侣设计的高颜值、轻量级自托管纪念日与生活点滴记录网站。

📌 **版本：v1.0.0** ｜ 🟢 **运行环境：Node.js v18+** ｜ 🚀 **核心框架：Express v4.19**

---

## ✨ 核心特色

1. **🌌 绝美视觉设计**
   * **毛玻璃美学 (Glassmorphism)**：采用半透明卡片、柔和的发光边框和多重高斯模糊层。
   * **动态环境微光**：背景中漂浮着可随时间缓慢律动的梦幻彩色微光光晕。
   * **暖心心跳动画**：大屏幕恋爱计时器配有自适应心跳波动微交互，仪式感满满。
2. **🔒 智能权限控制**
   * **双模式访问**：
     * **访客模式 (默认)**：无需输入密码，只能只读查看时间轴、浪漫回忆和照片，隐藏所有管理按钮。
     * **管理员模式**：点击锁按钮，输入管理员密码即可解锁。解锁后可在线**新增/编辑/删除纪念日、上传本地照片、一键备份/还原、修改站点标题与相爱起点日期**。
   * **后端安全防护**：所有的增删改查、图片上传及数据导入导出 API，均在服务器端进行签名 Cookie 校验，确保他人无法通过调用接口进行越权修改。
3. **📂 一键数据打包与相册归档 (ZIP)**
   * **无损全量备份**：一键导出包含 `database.json` 和 `uploads/` 下**所有真实图片照片**在内的 `.zip` 压缩包。
   * **一键全量恢复**：在任何时候、任何新服务器上，只需上传该备份 `.zip` 压缩文件，即可无缝重构历史相册和所有纪念日，完美解决了“文字备份了，图片丢了”的行业痛点。
4. **⚡ 极致轻量，零部署负担**
   * **文件型 JSON 数据库**：采用高性能文件级 JSON 引擎，**无需安装 MySQL/MongoDB 等数据库**，极易维护。
   * **零多余内存占用**：服务瞬间启动，资源消耗极低。

## 🛠️ 快速开始

### 1. 环境准备

确保您的电脑上已经安装了 [Node.js](https://nodejs.org/) (建议版本 v18+)。

### 2. 安装依赖

在项目根目录运行命令行工具，执行以下命令安装依赖：

```bash
npm install
```

### 3. 配置文件 `.env`

复制并重命名 `.env.example` 为 `.env`（项目中已为您配置好默认文件），您可以根据需要进行个性化调整：

* `PORT`: 运行的端口号（默认 3000）。
* `ADMIN_PASSCODE`: 管理员解锁密码（默认 `5201314`）。
* `COOKIE_SECRET`: Cookie 签名密钥，建议修改为更长、随机的字母组合。
* `TOGETHER_SINCE`: 恋爱计时器默认起点（格式为 YYYY-MM-DD）。
* `STATION_TITLE`: 小站的默认名称。

### 4. 启动服务

在终端运行：

```bash
npm start
```

控制台会打印出如下信息：

```text
====================================================
❤️  LoveStation is running on http://localhost:3000
🔑 Admin Passcode: 5201314
====================================================
```

此时在您的手机、电脑浏览器中输入 `http://localhost:3000` (如果部署在服务器上则是您的服务器 IP 和端口)，即可开启属于你们的爱之小站！

## 💾 数据备份与恢复指引

### 如何备份数据？

1. 输入密码解锁管理员模式。
2. 点击管理控制台上的 **【一键导出备份】**。
3. 浏览器会自动下载一个名为 `lovestation-backup-[时间戳].zip` 的文件，该文件已完整归档了您上传的所有相片和全部记录，请妥善保存。

### 如何在新设备上还原？

1. 在新的电脑或服务器上运行全新的 LoveStation 实例。
2. 解锁管理员模式。
3. 点击 **【一键导入恢复】**。
4. 选择您保存的 `lovestation-backup-[时间戳].zip` 文件，点击确定。
5. 系统会自动解压、重写数据库并重构相册文件夹，完成后页面会自动刷新，带您重回往昔美好！
